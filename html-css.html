Leçon 1 : Découverte du HTML

Qu'est-ce que le HTML ?

Le HTML (HyperText Markup Language) est le langage utilisé pour créer la structure des pages web.

Exemple simple :

<h1>Bonjour</h1>
<p>Bienvenue sur mon site.</p>

Résultat :

Bonjour

Bienvenue sur mon site.

Dans la prochaine leçon, vous apprendrez à utiliser les titres, les paragraphes, les images et les liens.
